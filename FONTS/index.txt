Directori: FONTS/

- Conte el codi font Java principal del projecte, drivers, persistencia i tests.

Visio general de la estructura important:
- domain/: Capa de domini amb les entitats principals
- persistence/: Implementacions de persistencia Java (PersistenceDriver, etc.).
- presentation/: Interficie d'usuari web (React + TypeScript) i NPM


Comandes utils:
- Compilar i executar:
	```pwsh
	cd FONTS
	make run
	```
- Compilar nomes (Java):
	```pwsh
	cd FONTS
	make jar
	```
